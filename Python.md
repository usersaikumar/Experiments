# Python
## How Python Program Executed
- Our python code gets executed line by line from the top
## Print Command
- **Print Numbers**
```
print(1234)
```
- **Print Variables**
```
print(a)
.
.
print(z)
```
- **Print Strings**
```
print('Name')
print("Name")
print('''Name''')
print("""Name""")
```
- **Print Operations**
```
print(1+2)
print(3-2)
print(4*3)
print(6/2)
print(8//3)
print(3**4)
```
